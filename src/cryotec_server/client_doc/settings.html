Настройки

  1. Адрес сервера - адрес сервера, на котором установленна серверная часть системы Обращайтесь к администратору
  2. Путь к api сервера - адрес api страницы сервера. Обращайтесь к администратору
  3. Название пакета сервера Обращайтесь к администратору
  4. Ваш логин - ваше имя пользователя
  5. Ваш пароль - ваш пароль
  6. Открывать ссылки из окна информации во внешнем браузере - если отмечено, ссылки на административную панель системы из окна информации будут открываться во внешнем браузере. Если нет, то они будут открываться внутри окна информации
  7. Стиль приложения - визуальный стиль приложения
  8. Восстановить значения - при нажатии на эту кнопку все настройки сбросятся на значения по умолчанию
  9. Тестировать соединение - тестировать соединение с новыми настройками
  10. Отмена - отменяет изменения настроек
  11. Сохранить - сохраняет настройки
